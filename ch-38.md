# Chapter 38
## 

| Original | English Translation |
| -: | -- |
| 上 德 不 德， |  |
| 是 以 有 德； |  |
| 下 德 不 失 德， |  |
| 是 以 無 德。 |  |
| 上 德 無 為 而 無 以 為， |  |
| 下 德 為 之 而 有 以 為。 |  |
| 上 仁 為 之 而 無 以 為。 |  |
| 上 義 為 之 而 有 以 為。 |  |
| 上 禮 為 之 而 莫 之 應， |  |
| 則 攘 臂 而 仍 之。 |  |
| 故 失 道 而 後 德， |  |
| 失 德 而 後 仁， |  |
| 失 仁 而 後 義， |  |
| 失 義 而 後 禮。 |  |
| 夫 禮 者， |  |
| 忠 信 之 薄 而 亂 之 首 也； |  |
| 前 識 者， |  |
| 道 之 華 而 愚 之 始 也。 |  |
| 是 以 大 丈 夫 處 其 厚， |  |
| 不 處 其 薄； |  |
| 居 其 實， 不 居 其 華。 |  |
| 故 去 彼 取 此。 |  |
